Set the Debug Mode

Consider you want to read the variable IS_RELEASE_SERVER from the standard input. If the variable is "true", it means it's a production server, and if "false", then it's a development server. Remember that the input is a string!

After you've read the variable, convert it to type bool.

Then, define the DEBUG variable: on the production server it should be set to False, and on the development server, it should be set to True.
